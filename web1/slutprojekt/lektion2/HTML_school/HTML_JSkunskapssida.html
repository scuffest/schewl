Java Script

1. Vad är JavaScript och varför är det viktigt för webbutveckling?

Gör hemsidor mer interaktiva och dynamiska.

2. Vad är ECMAScript och hur förhåller det sig till JavaScript?

Script som utvecklats baserat på ecma-262 standarder.

3. Vad är variabler i JavaScript och hur deklarerar du dem?

Variabler kan vara strängar, nummer, boolean värde och objekt. LET eller CONST. Let kan ändra värdena medans const är konstant och då omöjligt att skriva om. Let name =””; Let plays = true; Const x = 4000; Let y; y=true;

4. Vad är en funktion i JavaScript och hur definierar du en?

En funktion skapas för att göra en viss sak på en hemsidan. function function namn>(arg1,arg2,arg3,...) { Function code };

5. Vad är "strict mode" i JavaScript och varför kan det vara användbart?

Identifierar felaktig kod, förhindrar felaktiga variabler, förhindrar omdefiniering av nyckelord och reserverade ord, gör eval() säkrare, kan även öka prestanda ibland, förbättrar kodkvalitet. Aktiveras genom “use strict”;

6. Vad är skillnaden mellan "vanliga" JavaScript-bibliotek och ramverk som React, Angular och Vue?

Javascript bibliotek har fokus på enstaka funktioner, styckvis integration, “lätt” att lära och använda, Exempel på Javascript bibliotek jQuery, Lodash, Axios. Medan de andra har mer fokus på arkitektur och omfattande funktionalitet, och även en mer omfattande inlärningskurva.
